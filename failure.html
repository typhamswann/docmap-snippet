DocMap API Documentation

Endpoint: GET /v1/fake

This endpoint returns fake data for demonstration purposes. You can pass query parameters to filter or modify the response.

Example Request:

GET https://api.example.com/v1/docmapdev?status=active&limit=10

Example Response:

{
  "data": [
    { "id": 1, "status": "active", "info": "doc" },
    { "id": 2, "status": "active", "info": "map" }
  ],
  "count": 2
}
